# HTML-carousell
A simple Html carousell using JQUERY

Structure example:
```html
<div class="imgContainer">
  <div>
    <img src="1.jpg" class="active">
  </div>
  <div>
    <img src="2.jpg" class="disabled">
  </div>
</div>
```
